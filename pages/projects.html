Least-Cost Path Finder

A Python script that applies Dijkstra’s Shortest Path First algorithm to a CSV file representation of a graph. Each node in the graph represents a destination, and each edge represents a path. The algorithm finds the least costly path from a start point to each charging station in the graph (denoted by the charger symbol). This was a collaborative project with GitHub user ellieinfinity as the other contributer.
